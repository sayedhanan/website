# Getting Started with Next.js

Next.js is a powerful React framework that enables features like server-side rendering, static site generation, and more.

## Installation

To create a new Next.js app, run:

```bash
npx create-next-app@latest my-next-app
```

## File Structure

Next.js uses a file-based routing system:

```
my-next-app/
├── app/
│   ├── page.tsx        # Homepage (/)
│   ├── about/
│   │   └── page.tsx    # About page (/about)
│   └── blog/
│       └── [slug]/
│           └── page.tsx # Dynamic blog post page
├── components/
├── public/
└── package.json
```

## Adding Styling

Next.js supports various styling methods including CSS Modules, Tailwind CSS, and more.
